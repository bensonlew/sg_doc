split_fasta
==========================

模块Path
-----------

**tools.sequence.split_fasta**

功能描述
-----------------------------------

按序列条数切分fasta文件

参数设计
-----------------------------------

```
            {"name": "fasta", "type": "infile", "format": "sequence.fasta"},#输入fasta文件氨基酸序列
            {"name": "lines", "type": "int", "default": 100000},  # 序列数
```

运行逻辑
-----------------------------------
以序列ID计数，一条序列多行写为一行，序列数累加大于阈值则新建文件存入序列。

资源配置
-----------------------------------
```
        self._cpu = 2
        self._memory = '3G'
```
