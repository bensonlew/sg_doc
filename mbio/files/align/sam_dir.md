格式说明
==========================

Path
-----------

**align.bwa.sam_dir**


功能和用途描述
-----------------------------------

用于检查bwa软件生成的sam文件组成的文件夹,且文件下必须只能有'.sam'或'list.txt'结尾的文件


格式定义文档
-----------------------------------

pass

-----------------------------------

::
 ./
 ../
 sampele1.sam
 sampele.sam
 list.txt
